# 📊 Sales Data Analysis Dashboard <br>
## 📌 Project Overview <br>
This project focuses on analyzing sales data to uncover meaningful business insights and trends.
An interactive dashboard was created to monitor revenue, profit, and sales performance across regions and categories.
It enables stakeholders to make data-driven decisions by identifying growth opportunities and performance gaps. <br>
## 🎯 Objectives <br>
✔️ Analyze sales performance across different regions <br>
✔️ Identify top-selling products and categories <br>
✔️ Track monthly and yearly revenue trends <br>
✔️ Improve business decision-making using insights <br>
## 🛠️ Tools & Technologies <br>
🔹 Power BI / Excel / Python <br>
🔹 Data Cleaning & Transformation <br>
🔹 Data Visualization <br>
🔹 Dashboard Design <br>
